# マグネットシューター
  
## 概要説明
 専門学校に入学して約1ヶ月後に制作した作品です。<br>
 初のグループ制作作品で、メンバー全員がゲーム制作に慣れておらず、期間も限られていましたが、その中で全力を出して<br>
 完成させました。 <br>

## 開発期間  
 2024/5/27～2024/5/31<br>
  
## 開発環境：開発時に使用したツールや言語など  
 言語:C++<br>
 [Dxライブラリ](https://dxlib.xsrv.jp/)<br>

## 必要動作環境  
Windows10/11    

## 起動方法  
[Releasesページ](https://github.com/MasamichiKikuchi/MagnetShooter/releases)より実行ファイルをダウンロード  
→MagnetShooter.zipフォルダをダウンロード・解凍  
→MagnetShooter.exeを実行  
  
## 終了方法  
altキー & F4キー 同時押し


## 操作説明
［タイトル・ストーリー・説明・エンディング画面］<br>
 spaceキー・・・・・セリフ送り、次の画面<br>
<br>
［ゲーム画面］<br>
 十字キー・・・・・移動<br>
 Ⅹキー　・・・・・１手戻る<br>
 Ｃキー　・・・・・ステージリセット<br>
 Ｑキー　・・・・・操作方法表示<br>
